# 🛡 MaskForge 文件脱敏工具
适用于 AI 对话前的数据脱敏处理
一款本地运行的文件批量脱敏与还原工具。

支持对文档中的敏感词进行自动替换生成映射文件（JSON），
并在需要时一键还原原始内容。

---

## ✨ 功能特点

- ✅ 支持批量文件处理
- ✅ 支持拖拽上传
- ✅ 支持文件夹批量导入
- ✅ 自动生成 JSON 映射文件
- ✅ 一键还原脱敏内容
- ✅ 本地运行，不上传服务器
- ✅ 支持多种文档格式（docx / xlsx / pptx / txt 等）

---

## 🧩 工作流程

### 🔐 加密（脱敏）

1. 添加文件或文件夹
2. 输入敏感词（多个用英文逗号分隔）
3. 点击开始
4. 生成：
   - 脱敏后的文件
   - 映射文件（mapping.json）

---

### 🔓 解密（还原）

1. 将脱敏文件拖入解密区
2. 若自动未找到 mapping.json，可手动选择
3. 点击解密
4. 自动还原为原始内容

---

## ⚠️ 注意事项

- JSON 映射文件必须与脱敏文件匹配
- 删除映射文件将无法还原
- 建议加密与解密任务分开保存
